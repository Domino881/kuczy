### Installation:

#### with Vim-Plug:

add the line:

	Plug 'Domino881/kuczy'

to your plugins.vim file and run :PlugInstall

To enable, add:

	colorscheme kuczy

and

	let g:lightline = {
		\ 'colorscheme': 'kuczy',
		\}

to your .vimrc.
